# Skill: design-system-generator

## Purpose
Generate consistent UI/UX design guidelines and component systems.

## Description
This skill creates comprehensive design systems that ensure visual and functional consistency across the application. It defines colors, typography, spacing, components, and interaction patterns.

## Used By
- Frontend-Agent
- Main-Orchestrator

## Key Capabilities
- Define color palettes and themes
- Establish typography scales and hierarchies
- Create spacing and layout systems
- Design reusable component patterns
- Document interaction states (hover, active, disabled)
- Ensure accessibility standards (WCAG compliance)

## Usage Guidelines
- Start with design tokens (colors, spacing, typography)
- Create component specifications before implementation
- Ensure consistency with existing design patterns
- Document all component variants and states
- Consider mobile-first responsive design
