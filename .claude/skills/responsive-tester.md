# Skill: responsive-tester

## Purpose
Test frontend responsiveness, functionality, and user flows across devices.

## Description
This skill validates that frontend implementations work correctly across different screen sizes, devices, and browsers. It ensures responsive design, functionality, and user experience meet specifications.

## Used By
- Frontend-Agent
- Main-Orchestrator

## Key Capabilities
- Test layouts at multiple breakpoints
- Verify touch interactions on mobile
- Check navigation and routing
- Validate form submissions and validation
- Test authentication flows
- Verify API integration
- Check accessibility compliance
- Test error handling and edge cases

## Usage Guidelines
- Test at minimum breakpoints: mobile (375px), tablet (768px), desktop (1280px)
- Verify touch targets are at least 44x44px on mobile
- Test with browser developer tools device emulation
- Check keyboard navigation and screen reader compatibility
- Validate all user flows from start to finish
- Test error states and edge cases
- Verify loading states and transitions
- Check cross-browser compatibility
- Document any issues found with reproduction steps
