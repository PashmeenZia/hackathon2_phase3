# Skill: spec-validator

## Purpose
Ensure specifications are complete, testable, and follow conventions.

## Description
This skill validates that specifications meet quality standards and are ready for implementation. It checks for completeness, consistency, and adherence to project conventions.

## Used By
- Constitution-Keeper
- Spec Manager

## Key Capabilities
- Validate specification completeness
- Check for testable acceptance criteria
- Verify consistency across related specifications
- Ensure adherence to project conventions
- Identify missing requirements or edge cases

## Usage Guidelines
- Run validation before marking specifications as complete
- Check against constitution principles in `.specify/memory/constitution.md`
- Verify all placeholders are filled
- Ensure API contracts are fully defined
- Validate that database schemas include constraints and relationships
