# Skill: nextjs-page-generator

## Purpose
Generate frontend pages in Next.js following design and spec conventions.

## Description
This skill creates Next.js pages and components that follow project conventions, implement proper routing, handle authentication, and provide excellent user experience.

## Used By
- Frontend-Agent
- Main-Orchestrator

## Key Capabilities
- Create Next.js pages with proper routing
- Implement client and server components appropriately
- Integrate with authentication (Better Auth)
- Fetch data from backend APIs
- Handle loading and error states
- Implement responsive layouts
- Add proper TypeScript types

## Usage Guidelines
- Use App Router conventions (app directory)
- Implement server components for initial data fetching
- Use client components for interactivity
- Protect routes with authentication checks
- Handle API errors with user-friendly messages
- Implement loading skeletons for better UX
- Use TypeScript for type safety
- Follow component composition patterns
- Optimize for performance (lazy loading, code splitting)
