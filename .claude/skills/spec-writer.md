# Skill: spec-writer

## Purpose
Create clear, unambiguous, implementation-ready specifications.

## Description
This skill enables agents to write comprehensive specifications that serve as the single source of truth for feature implementation. Specifications must be testable, complete, and follow project conventions.

## Used By
- Spec Manager
- Main-Orchestrator

## Key Capabilities
- Write feature specifications with user stories and acceptance criteria
- Document API contracts with request/response schemas
- Define database schemas and relationships
- Create UI component specifications
- Ensure specifications are testable and measurable

## Usage Guidelines
- Follow the spec template in `.specify/templates/spec-template.md`
- Include explicit acceptance criteria for every requirement
- Document error cases and edge conditions
- Use concrete examples where applicable
- Ensure specifications are implementation-agnostic
