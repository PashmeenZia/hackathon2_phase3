# Skill: responsive-layout-designer

## Purpose
Design layouts that adapt to multiple screen sizes and devices.

## Description
This skill creates responsive layouts that provide optimal user experience across desktop, tablet, and mobile devices. It ensures proper breakpoints, flexible grids, and adaptive components.

## Used By
- Frontend-Agent
- Main-Orchestrator

## Key Capabilities
- Define responsive breakpoints (mobile, tablet, desktop)
- Create flexible grid systems
- Design adaptive navigation patterns
- Optimize touch targets for mobile
- Ensure content reflow and readability
- Handle orientation changes

## Usage Guidelines
- Use mobile-first approach
- Define clear breakpoints (e.g., 640px, 768px, 1024px, 1280px)
- Test layouts at various viewport sizes
- Ensure touch-friendly interactions on mobile
- Optimize images and assets for different screen densities
- Consider performance implications of responsive design
